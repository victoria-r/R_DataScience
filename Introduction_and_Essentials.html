# Functions
# Theme for plot
cool_theme <- function() {
  theme_minimal() +
  theme(
    text = element_text(family = "Bookman", color = "gray25"),
    plot.subtitle = element_text(size = 12),
    plot.caption = element_text(color = "gray30"),
    plot.background = element_rect(fill = "gray95"),
    plot.margin = unit(c(5, 10, 5, 10), units = "mm")
  )
}

Question 5

Type and execute the code ggplot(mtcars, aes(x=disp, y=mpg)) in R studio. Does the code display a chart? If so, explain the chart that was displayed; if the chart is not displayed, indicate what is missing and how would you update the code to show a scatterplot showing the relationship between disp and mpg.

ggplot(mtcars, aes(x=disp, y=mpg)) +
  cool_theme()

The code above displays a blank chart. To fix this you must add the geom_plot() function to the code. Below is the updated code showing the relationship between disp and mpg.

ggplot(data = mtcars) +
  geom_point(mapping = aes(x = disp, y = mpg)) +
  cool_theme()

Questions 6&7 will use the dataset mpg.

Question 6

Create a scatterplot showing the relationship between displ and hwy. Color each point based on the number of cylinders (cyl). Explain the scatterplot.

ggplot(data = ggplot2::mpg) +
  geom_point(mapping = aes(x = displ, y = hwy, color = cyl)) +
  cool_theme()

This scatter plot shows the relationship between three variables (hwy, displ, cyl) while still using a 2-D x,y plane.

Question 7

Copy the code for the scatter plot in question 6 and update it to apply facet_wrap using the drv column. What does the resulting charts indicate about rear wheel drive vehicles when compared to 4wd and front- wheel drive? Hint: use the help function to get more information on facet_wrap, e.g.?facet_wrap.

ggplot(data = ggplot2::mpg) +
  geom_point(mapping = aes(x = displ, y = hwy, color = cyl)) +
  facet_wrap(vars(drv)) +
  cool_theme()

The charts reveal that rear wheel drive vehicles have the lowest variety of number of cylinders and tend to have more engine displacement (in liters), when compared to front and 4 wheel drive vehicles.
